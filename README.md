
# 📊 Calculadora IRPF CGT VALENCIA COLEBEGA 2026

![Versión](https://img.shields.io/badge/versión-v1.0-blue)
![Android](https://img.shields.io/badge/platform-Android-green)

Una aplicación sencilla y precisa para calcular la retención del **IRPF (Impuesto sobre la Renta de las Personas Físicas)** aplicada a la nómina en España, actualizada con los tramos de 2026.

## 🚀 Funcionalidades
* **Cálculo Instantáneo:** Introduce tu salario bruto anual y obtén el resultado en segundos.
* **Desglose por Tramos:** Visualiza cuánto pagas en cada tramo impositivo.
* **Sueldo Neto:** Cálculo estimado del sueldo neto mensual (12 o 16 pagas).
* **Interfaz Limpia:** Diseñada para ser intuitiva y rápida de usar.
